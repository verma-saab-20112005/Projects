# Loan & Credit Risk Dataset Exploration

This repository contains an exploratory data analysis (EDA) notebook designed to analyze credit, loan, and demographic characteristics of borrowers from `original_dataset.csv`.

## 📌 Project Overview
The notebook performs initial data loading, inspects column data types, and computes frequency distributions for all categorical features within the dataset[cite: 19].

## 📁 Repository Structure
* **`notebook.ipynb`**: Jupyter notebook containing data loading and categorical feature analysis[cite: 19].
* **`README.md`**: Overview and execution instructions.
* **`OUTPUT.md`**: Summary of key statistical outputs and distributions[cite: 19].
* **`SUMMARY.html`**: A clean, formatted HTML report summarizing the findings[cite: 19].
