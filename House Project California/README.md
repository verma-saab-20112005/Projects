# California Housing Price Prediction Machine Learning Pipeline

## Overview
This project processes California housing block group data to train, evaluate, and deploy regression models for predicting median house values (`median_house_value`). The pipeline incorporates feature preprocessing, stratified train-test splitting based on income categories, cross-validation evaluation across multiple algorithms, and prediction generation on unseen test data.

## Dataset Features

* **`longitude`** *(float)*: Geographical longitude coordinate.
* **`latitude`** *(float)*: Geographical latitude coordinate.
* **`housing_median_age`** *(float)*: Median age of houses within a block group.
* **`total_rooms`** *(float)*: Total number of rooms within a block group.
* **`total_bedrooms`** *(float)*: Total number of bedrooms (contains 207 missing values in training set).
* **`population`** *(float)*: Total number of people residing within a block group.
* **`households`** *(float)*: Total number of households within a block group.
* **`median_income`** *(float)*: Median income for households in a block group (measured in tens of thousands of USD).
* **`ocean_proximity`** *(categorical)*: Location relative to the ocean (`<1H OCEAN`, `INLAND`, `NEAR OCEAN`, `NEAR BAY`, `ISLAND`).
* **`median_house_value`** *(float)*: Target variable representing median house value in USD.

---

## Machine Learning Pipeline Design

1. **Stratified Sampling**:
   * Data is split using `StratifiedShuffleSplit` (80% train / 20% test) based on binned income categories (`income_cat`) to avoid sampling bias.

2. **Feature Engineering & Preprocessing**:
   * **Numerical Pipeline**: Imputes missing values in `total_bedrooms` using median imputation (`SimpleImputer`), followed by z-score standardization (`StandardScaler`).
   * **Categorical Pipeline**: Encodes `ocean_proximity` using one-hot encoding (`OneHotEncoder`).
   * **Column Transformer**: Combines numerical and categorical transformations into a unified pipeline.

3. **Model Evaluation (10-Fold Cross-Validation)**:
   * **Linear Regression**: Mean RMSE ~ $69,204
   * **Decision Tree Regressor**: Mean RMSE ~ $69,081
   * **Random Forest Regressor**: Mean RMSE ~ $49,432 (Best Performing Model)
