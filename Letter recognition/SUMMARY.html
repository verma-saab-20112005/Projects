Dataset Overview

The Letter Recognition dataset contains pixel and geometric feature attributes generated from character displays representing 26 capital letters in the English alphabet. All features are integer-encoded numeric values representing spatial moments and edge statistics.

Key Insights

  • Well-Balanced Dataset: Class distribution is exceptionally uniform across characters (Max count: U with 813; Min count: H / Z with 734).
  • Clean Data Quality: All 20,000 instances are fully populated with zero missing values across all 17 features.
  • Feature Structure: Features capture bounding box dimensions (xbox, ybox, width, height), pixel counts (onpix), spatial averages (xbar, ybar), and edge/correlation attributes.
